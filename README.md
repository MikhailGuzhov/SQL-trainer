SQL Enterprise Arena: Core Infrastructure Preparation

Vision

Интерактивная среда для подготовки инженеров уровня Senior AQA / Data Architect. Проект эмулирует реальную базу данных крупного E-commerce холдинга и ставит перед пользователем 70 бизнес-критичных задач: от базовой аналитики до тяжелых DML-манипуляций и оптимизации запросов.

Технологический стек

Engine: AlaSQL (In-memory SQL database) — позволяет выполнять запросы прямо в браузере без задержек на сеть.

Frontend: React 18 (Standalone SPA) — современная компонентная архитектура.

Styling: Tailwind CSS — чистый, современный Enterprise-интерфейс с полной адаптивностью (Mobile/Desktop).

Persistence: LocalStorage API — сохранение прогресса прохождения и состояния сессии.

Архитектура базы данных (Enterprise Schema)

Система разворачивает 5 связанных сущностей:

Users: Клиентская база (Segment, LTV, Registration data).

Products: Справочник товаров (Category, Price, Stock).

Orders: Транзакционный слой (Status, Amounts).

Order_Items: Связующая таблица состава заказов (M2M).

Tickets: Слой поддержки пользователей (Priority, Issue tracking).

Основные разделы

1. База Знаний (Senior Specification)

Концентрированная техническая документация, включающая:

Execution Order: Порядок выполнения операторов (FROM -> WHERE -> ... -> SELECT).

NULL Anatomy: Нюансы работы с неопределенностью.

Performance: Разница между EXISTS и IN.

Join Mechanics: Глубокий разбор типов связей.

2. Брутальная Арена (Practice)

70 задач, разбитых на 15 уровней сложности. Каждая задача привязана к конкретному бизнес-кейсу (Маркетинг, Аналитика, Склад, Саппорт).

Установка и запуск

Проект является полностью автономным (Zero-dependency deployment).

Скачайте файл sql-arena.html.

Откройте его в любом современном браузере.

Весь функционал (БД, Редактор, Валидатор) готов к работе.

Цель проекта

Уничтожить страх перед сложными SQL-структурами и довести навык написания запросов до автоматизма. Проект не предлагает «пройти собеседование на мидла» — он заставляет тебя думать как владелец данных.
